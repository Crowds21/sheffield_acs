- 随机变量及其分布
  collapsed:: true
	- ((63308f48-142c-4d05-9183-ec18025d4125))
	  collapsed:: true
		- $F(x)=P(X\le x)$
			- $P(X<=x)$为当 X 小于一个特定值时,事件发生的概率
		- 这里的分布函数,是概率的总和
			- |X|a|b|c|
			  |P|P(a)|P(b)|P(c)|
			- X 代表所有可能发生的事件(a<b<c)
			- P 代表发生改事件的概率
			- F(c)就等于三者概率的总和
			- 是**概率函数**取值的**累加**结果，只是一种描述概率的不同手段
